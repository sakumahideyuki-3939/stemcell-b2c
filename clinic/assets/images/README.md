# clinic/assets/images/

写真・画像ファイルはここに格納してください。

## フォルダ構成

| フォルダ | 用途 |
|----------|------|
| `hero/` | ヒーロー背景写真（トップページ・下層ページ） |
| `blog/` | ブログ記事用写真（サムネイル・記事内画像） |

## 推奨仕様

- 形式: JPG（写真）/ PNG（図解・ロゴ）/ WebP（軽量化済み）
- ヒーロー画像: 幅1920px以上、容量500KB以下を目安に圧縮
- ブログサムネイル: 幅800px程度
- ファイル名: 英数字・ハイフン区切り（例: `hero-consult.jpg`, `blog-aga-thumb.jpg`）
